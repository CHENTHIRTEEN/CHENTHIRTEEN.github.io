Git使用注意及常用命令

Git使用注意及常用命令

git设置用户信息

不带后缀的命令可以查看本机用户信息

  • 将目录变为Git可管理的仓库

将文件添加到版本库

  • 将文件添加到仓库
  • 将文件提交到仓库
  • git命令必须在git仓库内执行
  • 注意添加文件时文件是否存在

时光机穿梭

  • git status命令可以查看仓库当前状态
  • 查看文件具体被修改的内容

git diff “filename”

版本回退

  • 查看历史纪录

git log从最近到最远显示提交日志

  • 带–pretty=oneline参数

  • git中HEAD表示当前版本上个版本为HEAD^N个版本为HEAD~N

  • 回退命令

  • 回退后再次使用git log
  • 窗口未关闭可以使用commit id回到某个版本

commit id取前几位即可

  • 若关闭了窗口可以通过git reflog查看记录的每一次命令
